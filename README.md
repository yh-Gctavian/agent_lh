## 项目概述

基于多因子打分的波段抄底策略系统，包含完整的数据层、因子计算、选股引擎、回测框架和胜率分析模块。

## 技术架构

```
数据层 → 因子层 → 选股层 → 回测层 → 分析层
```

## 因子权重配置

| 因子 | 权重 | 说明 |
|------|------|------|
| KDJ | 45% | 核心抄底指标，超卖信号 |
| 成交量 | 15% | 量比、缩量统计 |
| 均线 | 15% | 均线偏离度 |
| RSI | 10% | 相对强弱 |
| MACD | 10% | 趋势转折 |
| 布林带 | 5% | 通道位置 |

## 模块说明

### M1 数据层
- AKShare数据加载器
- 复权处理（前复权）
- ST/停牌/退市过滤
- Parquet缓存

### M2 因子层
- 6大因子计算
- talib实现 + 手动计算fallback
- 因子得分标准化

### M3 选股引擎
- 多因子加权打分
- 买卖信号生成
- 股票池过滤

### M4 回测框架
- 组合管理（单票10%，最大10只，总仓位80%）
- 订单撮合
- 费用计算（买入0.03%，卖出0.13%）
- 沪深300基准对比

### M5 胜率分析
- 核心指标：胜率/盈亏比/夏普/最大回撤
- 分层分析：年度/月度/市场环境
- 参数敏感性：网格搜索/Walk-Forward
- 报告生成：Markdown/HTML

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 运行回测
python wave_bottom_strategy/main.py --mode backtest --start 2020-01-01 --end 2025-12-31

# 参数优化
python wave_bottom_strategy/main.py --mode optimize

# 选股
python wave_bottom_strategy/main.py --mode select
```

## 风险提示

1. 策略基于历史数据回测，实盘表现可能有差异
2. 抄底策略在趋势下跌市场可能持续亏损
3. 建议配合止损策略使用
4. 实盘前建议小资金验证
