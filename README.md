# 波段抄底策略选股系统

基于多因子打分的波段抄底策略，完整实现因子计算、选股引擎、回测框架和胜率分析模块。

## 项目状态

| 阶段 | 状态 | 说明 |
|------|------|------|
| M0 项目初始化 | ✅ 完成 | 目录结构、配置框架 |
| M1 数据层 | ✅ 完成 | AKShare数据加载、复权处理、ST过滤 |
| M2 因子层 | ✅ 完成 | 6因子计算（KDJ/MA/成交量/RSI/MACD/布林带）|
| M3 选股引擎 | ✅ 完成 | 多因子打分、信号生成、股票过滤 |
| M4 回测框架 | ✅ 完成 | 回测引擎、组合管理、订单撮合、基准对比 |
| M5 胜率分析 | ✅ 完成 | 绩效指标、分层分析、参数优化、报告生成 |
| M6 联调优化 | ✅ 完成 | 主入口整合、文档完善 |

## 因子权重

| 因子 | 权重 | 说明 |
|------|------|------|
| KDJ | 45% | 抄底核心指标，超卖信号强 |
| 成交量 | 15% | 底部放量确认 |
| 均线 | 15% | 趋势辅助判断 |
| RSI | 10% | 超卖确认 |
| MACD | 10% | 趋势转折确认 |
| 布林带 | 5% | 价格通道辅助 |

## 仓位规则

- 单票最大仓位：10%
- 最大持仓数：10只
- 总仓位上限：80%

## 费用规则

- 买入佣金：0.03%
- 卖出费用：0.13%（含印花税0.1%）
- 滑点：0.1%

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 运行回测
python -m wave_bottom_strategy.main backtest --start 2020-01-01 --end 2025-12-31

# 执行选股
python -m wave_bottom_strategy.main select --top-n 10

# 参数优化
python -m wave_bottom_strategy.main optimize
```

## 项目结构

```
wave_bottom_strategy/
├── config/          # 配置模块
├── data/            # 数据模块（AKShare加载器）
├── factors/         # 因子模块（6因子实现）
├── selector/        # 选股模块（打分、筛选、信号）
├── backtest/        # 回测模块（引擎、组合、撮合）
├── analysis/        # 分析模块（指标、分层、优化）
├── utils/           # 工具模块
└── main.py          # 主入口
```
