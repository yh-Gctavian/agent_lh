## 二、技术方案对比

### 方案1: pytdx（推荐 ⭐）

**简介：** 开源Python库，直接读取通达信本地数据

**优点：**
- 完全离线，无需网络
- 支持读取通达信day文件夹
- 代码成熟，社区活跃
- 安装简单：`pip install pytdx`

**缺点：**
- 需要本地安装通达信
- 需要定期更新通达信数据

**示例代码：**
```python
from pytdx.hq import TdxHq_API

api = TdxHq_API()
if api.connect('119.147.212.81', 7709):
    data = api.get_security_bars(9, 0, '000001', 0, 100)
    # 9=日K线, 0=深市, 股票代码, 起始位置, 数量
    api.disconnect()
```

### 方案2: 通达信本地文件读取

**路径：** 通达信安装目录/vipdoc/sh/lday/ (上海) 或 sz/lday/ (深圳)

**文件格式：** .day 文件（二进制格式）

**解析方式：**
```python
import struct

def read_tdx_day_file(filepath):
    """读取通达信day文件"""
    with open(filepath, 'rb') as f:
        data = []
        while True:
            chunk = f.read(32)
            if not chunk:
                break
            # 解析32字节：日期, 开, 高, 低, 收, 成交额, 成交量
            date, o, h, l, c, amount, vol = struct.unpack('IIIIIII', chunk)
            data.append({
                'date': date,
                'open': o / 100.0,
                'high': h / 100.0,
                'low': l / 100.0,
                'close': c / 100.0,
                'volume': vol,
                'amount': amount
            })
        return data
```

```python
# data/tdx_loader.py
from pytdx.hq import TdxHq_API
import pandas as pd

class TdxDataLoader:
    def __init__(self, tdx_path=None):
        self.tdx_path = tdx_path
        self.api = TdxHq_API()
    
    def load_daily(self, code, start, end):
        """加载日K线"""
        # 0=深市, 1=沪市
        market = 0 if code.startswith(('0', '3')') else 1
        
        data = self.api.get_security_bars(
            category=9,  # 日K线
            market=market,
            code=code,
            start=0,
            count=1000
        )
        return pd.DataFrame(data)
```
