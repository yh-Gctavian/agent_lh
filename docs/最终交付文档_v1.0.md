## 一、项目概述

### 1.1 项目目标

开发一套完整的波段抄底策略选股系统，包含：
- 多因子计算（6因子加权打分）
- 选股引擎（抄底信号生成）
- 回测框架（2020-2025年历史回测）
- 胜率分析（绩效指标、分层分析、参数敏感性）

### 1.2 技术栈

- **语言：** Python 3.10+
- **数据源：** AKShare（免费无需token）
- **技术指标：** talib / 手动计算
- **存储：** Parquet（列式压缩）

## 五、核心功能

### 5.1 数据层

- `DataLoader`: AKShare数据加载器
- `DataProcessor`: ST/停牌过滤、复权处理
- `DataCache`: Parquet缓存管理

### 5.2 因子层

- `KDJFactor`: KDJ随机指标（核心抄底因子）
- `MAFactor`: 均线偏离度
- `VolumeFactor`: 量比、缩量天数
- `RSIFactor`: 相对强弱指标
- `MACDFactor`: 趋势转折确认
- `BollingerFactor`: 布林带位置

### 5.3 选股引擎

- `FactorScorer`: 多因子加权打分
- `StockFilter`: ST/停牌/退市过滤
- `SignalGenerator`: 买卖信号生成
- `SelectorEngine`: 选股引擎核心

### 5.4 回测框架

- `Portfolio`: 组合管理（持仓、资金、交易记录）
- `OrderMatcher`: 订单撮合（滑点、佣金、印花税）
- `Benchmark`: 基准对比（沪深300）
- `BacktestEngine`: 回测引擎核心

### 5.5 分析模块

- `PerformanceMetrics`: 绩效指标计算
- `LayeringAnalysis`: 分层分析（IC值）
- `SensitivityAnalysis`: 参数敏感性分析
- `ReportGenerator`: 报告生成（Markdown）

## 六、绩效指标

### 6.1 核心指标

| 指标 | 计算方式 |
|------|----------|
| 胜率 | 盈利交易 / 总交易 |
| 盈亏比 | 平均盈利 / 平均亏损 |
| 夏普比率 | (收益-无风险利率) / 波动率 × √252 |
| 最大回撤 | 峰值到谷底的最大跌幅 |
| 年化收益 | (1+总收益)^(252/天数) - 1 |

### 6.2 分层分析

计算因子IC值，验证因子有效性：
- IC > 0.05: 有效因子
- IC > 0.10: 强有效因子

## 七、参数优化建议

### 7.1 推荐参数范围

| 参数 | 推荐范围 | 说明 |
|------|----------|------|
| 止损 | 5%-10% | 防止单笔大亏 |
| 止盈 | 10%-20% | 适度止盈 |
| 持仓天数 | 3-7天 | 波段操作周期 |
| 仓位比例 | 10%-15% | 分散风险 |

### 7.2 资金规模建议

| 资金规模 | 建议持仓数 | 单只仓位 |
|----------|------------|----------|
| 10万以下 | 3-5只 | 15%-20% |
| 10-50万 | 5-8只 | 10%-15% |
| 50万以上 | 8-10只 | 8%-10% |
