## 三、BUG清单

### BUG-001: 股票池列名错误

**严重等级：** P0（阻塞测试）

**问题描述：**  
`load_stock_pool('hs300')` 使用列名 `'成分股代码'` 获取股票列表，但AKShare返回的列名可能为 `'成分券代码'`。

**错误日志：**
```
KeyError: '成分股代码'
```

**修复建议：**  
检查 `ak.index_stock_cons_weight_csindex(symbol='000300')` 返回的实际列名，修正代码。

---

### BUG-002: 交易日历日期类型比较错误

**严重等级：** P0（阻塞测试）

**问题描述：**  
`load_trade_calendar` 中日期比较时类型不匹配。

**错误日志：**
```
'>=' not supported between instances of 'datetime.date' and 'str'
```

**修复建议：**  
统一日期类型，比较前转换为同一格式。
